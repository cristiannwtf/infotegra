# 📌 Prueba Técnica Infotegra 🚀

Este repositorio contiene la prueba técnica para el ingreso a **Infotegra**. Se ha desarrollado utilizando **Laravel 12** y consume la API de **Rick and Morty**.

## 📌 Requisitos

- PHP 8.2+
- Composer
- MySQL o MariaDB
- Laravel 12
- XAMPP (opcional)
